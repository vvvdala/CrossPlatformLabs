## Запуск
Для того, щоб запустити лабораторну, варто виконати наступні команди з кореня лабораторної.

Для збірки проекту:
```bash
msbuild Build.proj /t:Build
```
Для запуску проекту:
```bash
msbuild Build.proj /t:Run
```
Для запуску тестів:
```bash
msbuild Build.proj /t:Test
```
